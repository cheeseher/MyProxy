# UI/UX 体验升级：汉化与图标美化计划

您希望在 Surge 面板中看到更直观的中文（如“直连”代替“DIRECT”）以及更优雅的图标（替换 Emoji）。

**遗憾的是：** Surge 的核心关键字（如 `DIRECT`, `REJECT`）是硬编码的保留字，**不能**直接改名为“直连”或“拒绝”。但是，我们可以通过**创建一个中文命名的策略组别名**来“曲线救国”。

## 1. “直连”汉化方案

我将创建一个名为 **`🎯 全局直连`** 的策略组，它本质上就是 `DIRECT`，但显示在面板上就是中文。

*   **策略组**: `🎯 全局直连 = select, DIRECT`
*   **应用**: 将所有原本指向 `DIRECT` 的策略组选项，替换为这个新的 `🎯 全局直连`。

## 2. 图标美化方案 (Icon-Set)

Surge 支持远程图标集 (`icon-url`)。我将为每个策略组配置一套统一、优雅的图标（通常来自 GitHub 上流行的彩色图标库，如 `Koolson` 或 `Semporia`），替换掉目前的 Emoji。

*   **GitHub**: 使用 GitHub Logo 图标
*   **Telegram**: 使用 Telegram 官方图标
*   **Zoom**: 使用 Zoom 官方图标
*   **Netflix/YouTube**: 使用各自的品牌图标
*   **地区节点**: 使用精美的国旗图标

## 3. 配置文件修改预览

我将修改 `Surge-Dual.conf`：

1.  **[Proxy Group]**:
    *   新增 `🎯 全局直连` 组。
    *   为所有策略组添加 `icon-url=...` 参数。
    *   将所有策略组中的 `DIRECT` 替换为 `🎯 全局直连`。

准备就绪，请确认此计划，让您的 Surge 面板焕然一新。
